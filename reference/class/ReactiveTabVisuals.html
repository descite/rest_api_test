Glia JS SDK » Index » ReactiveTabVisuals

Class: ReactiveTabVisuals

Property Summary

(string) backColor

Back color of the Reactive Tab as a hex code.

(string) frontColor

Front color of the Reactive Tab as a hex code.

(number) fontSize

Font size of the Reactive Tab. One of 20, 24, 27.

(string) format

Whether operators' available medias should be displayed in text or in both text and icons. Possible values are 'text' for only text and 'mix' for both text and icons.

(string) iconType

The icon to be shown in the Reactive Tab. Possible values are 'audio', 'text' and 'video'.

(string) placement Deprecated

Shows on which side the Reactive Tab should be placed. When 'right', the Reactive Tab is placed to the right side of the web page. When 'left', the Reactive Tab is placed on the left side of the page.

Deprecated. Use panelPosition instead.
(number) size

The scale of the Reactive Tab as a percentage of the default size. Suggested values are between 80 and 100.

(string) text

Text to be displayed on the Reactive Tab.

(number) verticalOffset

The vertical offset of the Reactive Tab from the bottom of the browser’s viewport, expressed as a percentage. Possible values range from 0 to 100.
